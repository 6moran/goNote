## 单个路由中间件

```go
//绑定多个函数，中间的是中间件
router.GET("/", func(c *gin.Context) {
    
}, func(c *gin.Context) {
    
})
```



```go
//终止请求，执行到c.Abort下面绑定的中间件或逻辑就不执行了，但是会执行完当前的响应中间件
//注意：c.Abort只是停止了接下来的中间件或逻辑，当前中间件还是要执行完毕的
c.Abort

//例子
router.GET("/", func(c *gin.Context) {
    fmt.Println("m1")
    c.Abort()
    fmt.Println("m3")
}, func(c *gin.Context) {
    fmt.Println("m2")
})
//会打印出m1和m3


//响应中间件设定
c.Next
//只要代码写在c.Next之下的就是响应中间件，响应中间件是倒着的顺序执行的
```

## 全局中间件

```go
router.Use(func(c *gin.Context) {
		fmt.Println("m2")
	}, func(c *gin.Context) {
		fmt.Println("m3")
	})

//这样写每次注册路由都会有这个中间件
```









## 中间件传值

```go
//中间件中写
c.Set("name", "付涛")

//获取中写
name, _ := c.Get("name")
//这个是any类型，需要断言才能用
fmt.Println(name.(string))
```
