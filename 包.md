### 包的导入：

```
import (
    "day_start/test"
    "fmt"
)
```

导包只需要导入src后面路径，\换成/



**自定义包名**

一个目录下的同级文件归属一个包

（package aaa，则一个目录下全都是aaa）



**可以对导包路径起别名**

```
import (
    te "day_start/test"
    "fmt"
)
fmt.Println(te.StuNo)
```
