# 文件上传

## 一、单文件

```go
router.POST("/upload", func(c *gin.Context) {
    file, _ := c.FormFile("file")
    fmt.Println(file.Filename)
    fmt.Println(file.Size / 1024)                         //file.Size单位是字节（B），除以1024为KB
    c.SaveUploadedFile(file, "static/img/"+file.Filename) //保存到"static/img/"路径下
    c.JSON(200, gin.H{
       "msg": "上传成功",
    })
})
```



## 二、多文件

```go
router.POST("/uploads", func(c *gin.Context) {
    form, _ := c.MultipartForm()
    files, _ := form.File["upload[]"]	//这里的upload对应前端的名字
    for _, file := range files {
       c.SaveUploadedFile(file, "static/img/"+file.Filename)
    }
    c.JSON(200, gin.H{"msg": fmt.Sprintf("成功上传%d个文件", len(files))})
})
```



## 三、服务端保存文件的方法

### 一、SaveUploadedFile

```go
c.SaveUploadedFile(file, "static/img/"+file.Filename) //保存到"static/img/"路径下，名称为file.Filename
```

### 二、Create+Copy

```go
reader, _ := file.Open()
writer, _ := os.Create("static/img/" + file.Filename)	//如果文件已存在，会清空内容（覆盖）
n, _ := io.Copy(writer, reader) //n表示一共拷贝了多少字节
```









# 文件下载

```go
router.GET("download", func(c *gin.Context) {
    //要下载的文件路径
    filePath := "static/img/5ea652640838e8fc12000675-1.jpg"
    //取到文件名
    fileName := filepath.Base(filePath)
    //设置文本类型为文件流，唤起浏览器的下载
    c.Header("Content-Type", "application/octet-stream")
    //一般设置了这个就需要设置下载的文件名
    c.Header("Content-Disposition", "attachment;filename="+fileName)
    //设置传输过程的编码形式，解决乱码的情况，也可以不设置
    c.Header("Content-Transfer-Encoding", "binary")
    //返回一个文件
    c.File(filePath)
})
```

**注意：**前后端模式下，后端只需要返回一个文件数据，前端来唤起下载，文件名写在响应头上
