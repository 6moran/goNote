**闭包：**闭包的本质依旧是一个匿名函数，只是这个函数引入外界的变量/参数

*（匿名函数+引用的变量/参数=闭包）*



匿名函数所引用的那个变量会一直保存在内存中，不断使用



```go
// 闭包函数，getSum的形参为空，返回值是一个匿名函数，这个函数的形参是int类型，返回值也是int类型
func getSum() func(int) int {
    var sum int = 0
    return func(num int) int {
       sum = sum + num
       return sum
    }
}

f1 := getSum	//将函数getSum赋值给f1
f2 := f1()		//调用f1函数将返回值函数赋值给f2
fmt.Println(f2(1)) //sum=0,输出1
fmt.Println(f2(2)) //sum=1,输出3
fmt.Println(f2(3)) //sum=3,输出6
//sum随着匿名函数的引用不断改变
```

闭包不可以滥用

使用闭包的情况：保留上次引用的某个值
