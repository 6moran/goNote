### 函数：

- 不能写到main函数中
- 首字母大写为public，首字母小写为private
- 形参和返回值可以为0，1，n个

**代码格式**

```go
func add(num1 int, num2 int) int {     //如果返回值为一个的话，括号可以省略
    var sum int = num1 + num2
    return sum
}
```

```go
func add(num1 int, num2 int) (int, int) { //如果返回值为一个的话，括号可以省略
    var sum int = num1 + num2

    var result int = num1 - num2
    return sum, result
}
```



***Go函数不支持重载***



**可变参数arg：**

```go
// 可变参数，可以传一个或者多个参数
func test(arg ...int) {
    for i := 0; i < len(arg); i++ {
       fmt.Println(arg[i])
    }
}

test()
test(1)
test(7, 5, 6)
```

**想利用函数进行改变main函数中的值，运用指针**

```go
func sort(num1 *int, num2 *int) {
    var t int = *num1
    *num1 = *num2
    *num2 = t

}

var num1 int = 1
var num2 int = 2
fmt.Printf("交换前num1=%d，num2=%d\n", num1, num2)
sort(&num1, &num2)		//指针传的是地址
fmt.Printf("交换后num1=%d，num2=%d\n", num1, num2)
```



**函数也是一种数据类型，可以赋值给变量**

```go
func test2(num int) {
    fmt.Println(num)
}
//函数也是一种数据类型，可以赋值给一个变量
var a = test2
fmt.Printf("a的类型为：%T，test函数的类型为：%T \n", a, test2)
a(10)
```



**函数也可以为形参**

```go
func test3(num int, num2 float32, testFunc func(int)) {
    fmt.Println("-------test3--------")
}
//函数也可以为形参
var a = test2
test3(10, 3.14, test2)
test3(10, 3.14, a)
```



**GO支持自定义数据类型**

```go
//type mySum func(int,int)int  这时mySum就等价一个函数类型func(int,int)int
//给func(int)一个别名
type myFunc func(int)
func test4(num int, num2 float32, testFunc myFunc) {
    fmt.Println("-------test3--------")
}
test4(10, 4.5, a)

//相当与给int起了个别名myInt，但是int和myInt不是一个数据类型
type myInt int
var n1 myInt = 30
fmt.Printf("n1=%d\n", n1)

```



**GO支持对函数返回值命名**

```go
func add2(num1 int, num2 int) (sum int, sub int) { //如果返回值为一个的话，括号可以省略
    sum = num1 + num2
    sub = num1 - num2
    return    //可以省略返回值不写
}
var n2, n3 = add2(num1, num2)
fmt.Printf("n2的值为%d，n3的值为%d\n", n2, n3)
```
