## 路由分组

```go
api := router.Group("api")
api.GET("/index", func(c *gin.Context) {
    c.String(200, "index")
})
api.GET("/home", func(c *gin.Context) {
    c.String(200, "home")
})
```



## 路由分组嵌套

```go
r := router.Group("api")
//再分一次组
userManger := r.Group("user_manger")
{
    userManger.GET("/users", func(c *gin.Context) {
       c.String(200, "userManger")
    })
}
articleManger := r.Group("article_manger")
{
    articleManger.GET("/articles", func(c *gin.Context) {
       c.String(200, "articleManger")
    })
}
```

通常这两个嵌套分组可以单独提出来写一个函数

```go
func UserRouterInit(r *gin.RouterGroup) {
    userManger := r.Group("user_manger")
    {
       userManger.GET("/users", func(c *gin.Context) {
          c.String(200, "userManger")
       })
    }
}

func ArticleRouterInit(r *gin.RouterGroup) {
    articleManger := r.Group("article_manger")
    {
       articleManger.GET("/articles", func(c *gin.Context) {
          c.String(200, "articleManger")
       })
    }
}
```



## 路由分组定义中间件

可以对整个组Use一个中间件，即定义一个路由组的全局中间件

```go
func UserRouterInit(r *gin.RouterGroup) {
    fmt.Println("哈哈哈")
    //这里绑定了一个Middleware()函数，它的的返回值是中间件
    userManger := r.Group("user_manger").Use(Middleware())
    {
       userManger.GET("/users", func(c *gin.Context) {
          c.String(200, "userManger")
       })
    }
}

func Middleware() gin.HandlerFunc {
    //这个是个函数，绑定中间件时就打印了
    fmt.Println("外面...")
    //这里是真正的逻辑，在收到请求时触发
    return func(c *gin.Context) {
       fmt.Println("里面...")
    }
}
```

```go
//当然，这种模式也可以使用闭包
func Middleware(msg string) gin.HandlerFunc {
	//这个是个函数，绑定中间件时就打印了
	fmt.Println("外面...")
	//这里是真正的逻辑，在收到请求时触发
	return func(c *gin.Context) {
		fmt.Println("里面...")
		c.JSON(200, gin.H{"msg": msg})
		c.Abort()

//在路由函数中引入
func UserRouterInit(r *gin.RouterGroup) {

		userManger.GET("/users", func(c *gin.Context) {
			c.String(200, "userManger")
		})
	}
}
```





## gin.New()和gin.Default()的区别

gin.New()相当于打开了一个裸引擎，不包含任何中间件，需要调用者自己去完善，更灵活点。

gin.Default()是gin使用gin.New()创建的引擎，里面默认注册了两个中间件，Logger（日志中间件）和Recovery（恢复中间件），更好用，不容易崩服务。
