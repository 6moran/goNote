### 方法：

​	golang中的方法是作用在指定的数据类型上的（即：与指定的数据类型绑定），自定义类型都可以有方法。

**结构体：**

```go
type Dog struct {
    Age  int
    Name string
}

func (dog Dog) test() {
    fmt.Println("狗的方法")
}
```

只有Dog结构体的实例能够调用，方法中可以直接调用dog的属性。

**自定义数据类型：**

```go
type integar int

func (i integar) print() {
    fmt.Println("i=", i)
}
```



**方法的注意事项和细节：**

1.结构体是值拷贝，在方法中调用并不会改变main函数中结构体的值。

2.在方法中，如果某个结构体写了String方法，那么输出时函数会自动调用String方法进行输出。

```go
func (per Person) String() string {
    str := fmt.Sprintf("Name=%v,Age=%v", per.Name, per.Age)
    return str
}
p := Person{"fugang", 18}

```
